# Job Portal Web Application

A responsive and user-friendly job portal that streamlines the hiring process, enabling job seekers to explore opportunities and recruiters to post and manage job listings through a clean web interface powered by a robust Java backend.

## Key Features
- Role-based authentication for Job Seekers and Recruiters  
- Recruiter job posting and management  
- REST API integration for seamless client-server communication  
- PostgreSQL database support with Hibernate ORM  
- Project built using Maven for structured dependency and build management  
- Fully responsive frontend using HTML, CSS, and JavaScript  

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java, Spring Boot, Spring Security, Hibernate  
- **Database:** PostgreSQL  
- **Build Tool:** Maven  

## Application Flow
1. Recruiter posts a new job using backend APIs  
2. Job seekers browse all available jobs  
3. Job seeker submits application via API  
4. Recruiter fetches applicants list using job ID
